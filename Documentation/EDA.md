# Week 3: Exploratory Data Analysis (EDA)

Date:- 12-03-2026


Exploratory Data Analysis (EDA) is the process of analyzing datasets to summarize their main characteristics using statistics and visualizations.

The main objectives of EDA are:

- Understanding the dataset
- Handling missing values
- Handling outliers

# Handling Missing Values:
Missing values occur when some data entries are not recorded. These values must be handled before performing analysis.

Common techniques include:
- Removing rows with missing values
- Filling missing values using mean, median, or mode

# Handling Outliers:

Outliers are extreme values that fall outside the normal range of the data.

Two types of outliers:
1. Values lower than the expected range   Ex: Age = -15
2. Values higher than the expected range  Ex: Age = 1000

Outliers can distort analysis results and therefore need to be detected and treated.

# Types of Data

There are three main types of data:

1. Structured Data
Data organized in rows and columns.

Types of structured data include:

i) Categorical Data
  - Example: Names, Weather conditions

ii) Numerical Data
  - Example: Age, Number of vehicles involved

iii) Time Series Data
  - Example: Date and Time

2. Unstructured Data
Data without a predefined format.

Example:
- Images
- Videos
- Text documents

3. Semi-Structured Data
Data that does not follow a strict tabular format but still contains organizational properties.

Example:
- JSON files
- XML files
